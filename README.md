# PergunTalys - Jogo de Raciocínio Lógico

PergunTalys é um jogo educativo desenvolvido para estimular o raciocínio lógico e introduzir conceitos iniciais de programação para o público infanto-juvenil.

Acesse o jogo: https://thiago-stein.github.io/jogo-perguntalys/

---

## O Mascote Talys
O jogo conta com a presença do Talys, um axolote rosa que acompanha o jogador durante toda a jornada, tornando a experiência de aprendizado mais amigável.

## Sobre o Projeto
O PergunTalys conta com 10 fases de dificuldade progressiva. Cada etapa foi planejada para preparar o usuário para os desafios seguintes, resultando na fase final, que exige maior capacidade analítica e aplicação dos conceitos trabalhados anteriormente.

### Características do Jogo
* Público-alvo: Crianças e adolescentes.
* Objetivo: Desenvolvimento de lógica de programação e resolução de problemas.
* Persistência de Dados: O progresso do usuário é salvo de forma local, permitindo que a pontuação e as fases alcançadas sejam mantidas entre as sessões.

## Tecnologias Utilizadas
* HTML5: Estruturação da interface.
* CSS3: Estilização visual.
* JavaScript: Implementação da lógica do jogo, controle de fluxo e manipulação de eventos.
* LocalStorage: Armazenamento de dados do progresso do jogador diretamente no navegador.
