# PergunTalys - Jogo de Raciocínio Lógico

PergunTalys é um jogo educativo desenvolvido para estimular o raciocínio lógico e introduzir conceitos iniciais de programação para o público infanto-juvenil.

Acesse o jogo: https://thiago-stein.github.io/jogo-perguntalys/

---

## Desenvolvimento Artístico
Para nosso jogo, queríamos trazer um visual, colorido, divertido e confortável para os jogadores. Com isso, decidimos em uma paleta de cores que mistura tons 
pastéis de rosa, amarelo e ciano para as bordas da tela, os balões de fala, títulos e padrões. Já para o fundo, optamos por um azul celeste um pouco mais escuro,
que destacasse as cores mais claras e as cores do Talys em si, fazendo dele o ponto focal da tela. Para manter a atmosfera amigável que criamos com o design da 
interface do jogo, decidimos utilizar tipografias mais arredondadas e de fácil leitura, como a fonte Baloo Phai 2 para os textos curtos e de maior destaque (título
do jogo, das fases e da tela final), e a fonte Sour Gummy para os textos mais longos (descrição das fases e dicas).


## O Mascote Talys
Na fase inicial de planejamento da interface do jogo, decidimos adicionar um mascote para fazer as perguntas e acompanhar o progresso do jogador. Depois de 
percorrer uma longa lista de opções de animais que poderiam ser a identidade do jogo, escolhemos um animal aquático não muito comum. Foram muitos rascunhos, testes
e conversas até chegarmos em um design que se encaixasse com o que tínhamos em mente. Assim foi criado o Talys, um simpático axolote rosa de olhos azuis que era 
perfeito para nosso jogo, pois ele não só se encaixava perfeitamente com a paleta de cores, como também tem os traços arredondados, fofos e acolhedores que 
estávamos buscando.

## Sobre o Projeto
O PergunTalys conta com 10 fases de dificuldade progressiva. Cada etapa foi planejada para preparar o usuário para os desafios seguintes, resultando na fase final, 
que exige maior capacidade analítica e aplicação dos conceitos trabalhados anteriormente.

### Características do Jogo
* Público-alvo: Crianças e adolescentes.
* Objetivo: Desenvolvimento de lógica de programação e resolução de problemas.
* Persistência de Dados: O progresso do usuário é salvo de forma local, permitindo que a pontuação e as fases alcançadas sejam mantidas entre as sessões.

## Tecnologias Utilizadas
* HTML5: Estruturação da interface.
* CSS3: Estilização visual.
* JavaScript: Implementação da lógica do jogo, controle de fluxo e manipulação de eventos.
* LocalStorage: Armazenamento de dados do progresso do jogador diretamente no navegador.
